Aurora Hak Explorer 0.2.2 — Windows x86-64 portable build

No installation is required. Run Aurora-Hak-Explorer.exe.

This build is intended for Windows 10 or Windows 11 on a 64-bit Intel or AMD
processor. It contains the Microsoft C/C++ runtime statically, so no separate
Visual C++ Redistributable should be needed.

Please test with a COPY of an archive first:

1. Open, search, sort, and switch between multiple HAK tabs.
2. Add files with the picker, drag-and-drop, and Ctrl+V.
3. Copy, cut, export, and drag resources from Aurora into Explorer.
4. Save, close, reopen, and verify the resource names and counts.
5. Check overwrite prompts and the unsaved-changes prompt.
6. Enable View > Compact mode, restart Aurora, and confirm it is remembered.

If something fails, please record the Windows version, the operation being
performed, and the exact error message or a screenshot.
